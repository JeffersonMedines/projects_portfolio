Previsão de vendas para as próximas 6 semanas das lojas Rossmann.

Nesse projeto foi desenvolvido um modelo de regressão utilizando os algoritmos de Machine Learning Random Forest e XGBoost para resolver um problema de previsão de vendas para as próximas 6 semanas de uma rede de farmácias da Europa. O problema de negócio era o CFO conseguir definir um budget para a reforma das lojas de acordo com a receita que cada uma irá gerar nas próximas 6 semanas, se baseando em previsões mais acertivas e escaláveis. O modelo final realizou previsões com erro percentual médio de 10% e prevendo uma receita de R$ 282mi no total. O deploy do modelo em produção pode ser consultado tanto por uma requisição em API, e também por qualquer smartphone com o aplicativo telegram instalado.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy, Seaborn, Scipy, Matplotlib, Pickle e Scikit-Learn.
  • Jupyter Notebook e VSCode.
  • Telegram Bot.
  • Heroku Cloud e Flask.
  • Random Forest, XGBoost e Boruta.
  • Git e Github para versionamento de código.
